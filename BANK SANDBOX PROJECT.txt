Project to Start: Bank API Sandbox

Definition:
A simulated backend banking system designed to mimic how real-world financial institutions manage accounts, 
balances, and transactions. Similar to a real bank but Known as a safe testing environment that behaves like one.

Core Purpose of project
===========================
Provides developers with:
1.	Vitual Users, can create customers who hold bank accounts in the system.
2.	Virtual bank accounts, each user account has a balance currency unique account number & timestamps
3.	Bank operations, support despositing money, withdrawing, transfering (bank-to-bank transfers) like in real banks apps etc

Transaction Ledger
===========================
Every operation iss recorded as a transaction as
-	transaction_id
-	from_account
-	to_account
-	type (deposit/withdraw/transfer)
-	amount
-	status
-	timestamp
- 	Fraud Detection

This builds an audit trail â€” EXACTLY how banks track money.



API-Driven System
==========================
All interactions happen through REST API endpoints, such as:
-	POST /users
-	POST /accounts
-	POST /accounts/{id}/deposit
-	POST /transactions/transfer

This teaches you to design, build, test, and secure financial APIs.
